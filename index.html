Css selectors

And the use of them

There are a lot of different css selectors. Each are used for different things, depending on what you want to style.

A type selector is the most general one. For example, using the div tag will select and style every div element. With ID's and classes, you can make it more specific. An ID selector may only have one ID tag and the ID tag can only be use once in the HMTL doc. This differs from the class selector, which may be used multiple times. An element may contain several diffrerent classes.

Short commandos in VSC

There are several different short commandos in Visual Studio Code that makes it your coding more efficient. My top three favorites are:

  1. Ctrl + D: Multimakrs identical code
  2. Alt + up/down: Moves code up or down
  3. Alt + shift + down: Copies the code and places it below

Blocks vs inline block

Block

Display:block;

Starts on a new line and takes up the whole width. (dominating!). Forces elements to be placed beneath. Is effected by properties that manipulate width/height.

Inline block

Display:inline-block;

Doesn't take up the whole row and allows elements to be placed beside it. An inline-block if not effected by the height property.

Flexbox

Flexbox is a modern and efficient way of creating layouts. May well be used in combination with CSS grids.

CSS grids

A modern and efficient way of creating a layout by lining up a grid and creating columns and rows. Differs from flexbox since it's two-dimensional
